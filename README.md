 ## Crea un repositorio en GitHub
1. Vas GitHub.com inicias sesión o te registras.

2. Luego de estar dentro de tu cuenta de Git. Vas a repositorios, que lo puedes hallar en la parte superior 
o lo puedes encontrar en la esquina de las tres líneas.

3. Presionas "repositorio", y luego creas uno en "nuevo repositorio".

4. Luego ponle un nombre, lo configuaras y listo. Ya tendrías un respositorio.
